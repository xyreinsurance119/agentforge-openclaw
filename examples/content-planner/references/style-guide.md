# Гайд по стилю контента

## Тон

- Дружелюбный, но экспертный
- Без канцелярита ("осуществляем создание контента" -> "делаем контент")
- Без воды ("в современном мире" -> удалить)
- Конкретика важнее красоты

## Структура поста

1. **Крючок** - первое предложение цепляет (вопрос, факт, провокация)
2. **Контекст** - зачем это важно (2-3 предложения)
3. **Мясо** - основная ценность
4. **Призыв** - что делать дальше (без "подписывайтесь на мой канал")

## Заголовки

- Конкретные числа: "3 инструмента" лучше чем "несколько инструментов"
- Без кликбейта: обещай только то, что есть в посте
- Длина: 5-10 слов

## Платформы

| Платформа | Длина | Особенности |
|-----------|-------|-------------|
| Telegram | 300-800 знаков | Без хештегов, минимум эмодзи |
| YouTube | 10-15 мин (средне) | SEO заголовок, тайм-коды |
| Twitter/X | 280 знаков / тред | Первый твит = крючок |
| Instagram | 150-300 слов | Визуал первичен |

## Частота

- Лучше 2 качественных поста в неделю, чем 7 средних
- Регулярность важнее количества
- Если нечего сказать - не пиши ради галочки
